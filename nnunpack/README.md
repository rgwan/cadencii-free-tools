Niao Niao Soundbank unpack Tool
=====================

Extract Wave file from NiaoNiao Soundbank.

BUILD
-------
`make`

USAGE
-------

unpack <inf.d> <voice.d> <extract directory>

CAUTION: <extract directory> have to be exist!!!
